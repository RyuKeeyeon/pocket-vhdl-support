Pocket VHDL is an offline editor for creating and editing .vhd and .vhdl documents on iPhone and iPad.

Features

  • Create, open, and save VHDL documents through the Files interface
  • Syntax highlighting for keywords, types, strings, numbers, and comments
  • Completion for declared signals, variables, types, and VHDL language items
  • Find, replace, go to line, and symbol outline
  • Automatic indentation, paired characters, and hardware keyboard shortcuts
  • Portrait and landscape support on iPhone and iPad

Quick start

Create a document

Launch the app, choose a new document, and start typing. Select the filename and location in the system document interface.

Open an existing document

Select a .vhd or .vhdl file from the Files app or Pocket VHDL's document browser.

Use code completion

After two or more characters, suggestions appear from declarations in the current document and built-in VHDL items. Tap a suggestion, or select one with the arrow keys and insert it with Tab on a hardware keyboard.

Find and navigate

Use the search button for find and replace. Open the symbol outline to navigate to entities, architectures, processes, and declarations.

Frequently asked questions

Does Pocket VHDL compile or simulate code?

No. The current version focuses on document editing and does not provide compilation or simulation.

Are my documents uploaded to a server?

No. Pocket VHDL does not collect document contents or send them to a developer-operated server. Documents remain in the local or file-provider location you select.

Which file formats are supported?

Pocket VHDL supports plain-text .vhd and .vhdl documents.

I cannot open or save a file.

First confirm that the location and file are accessible in the Files app. For a cloud file, wait for its download to finish and try again. A read-only file may need to be duplicated or moved to a writable location.

How do I restore the original editor settings?

Open Editor Settings from the More button at the top right of the editor, then choose Reset to Defaults. This does not delete document contents.

Troubleshooting

  1. Save the document you are editing, then reopen Pocket VHDL.
  2. Check Settings for the latest iOS or iPadOS update available to your device.
  3. If the issue affects only one file, check whether it also occurs in a minimal example with confidential content removed.

For a serious issue such as possible document loss or repeated crashes, keep the original file unchanged, make a copy, and contact support.

Contact

When reporting a problem, include the Pocket VHDL version, device model, OS version, actions that caused the issue, and steps to reproduce it. Do not attach confidential or work-related VHDL source code, passwords, or personal identifiers.
