코드는 가볍게,
VHDL은 어디서나.

Pocket VHDL은 iPhone과 iPad에서 .vhd 및 .vhdl 문서를 만들고 편집하는 오프라인 코드 편집기입니다.

주요 기능

  • 파일 앱을 통한 VHDL 문서 생성, 열기 및 저장
  • 키워드, 타입, 문자열, 숫자 및 주석 구문 강조
  • 선언된 신호, 변수, 타입과 VHDL 키워드 자동완성
  • 찾기, 바꾸기, 줄 이동 및 코드 개요
  • 자동 들여쓰기, 괄호 자동 닫기 및 외장 키보드 단축키
  • iPhone·iPad 세로 및 가로 화면 지원

빠른 사용법

문서 만들기

앱을 실행하고 새 문서를 선택한 다음 코드를 입력하세요. 저장 위치와 파일명은 iOS 또는 iPadOS의 파일 화면에서 선택합니다.

기존 문서 열기

파일 앱이나 Pocket VHDL의 문서 선택 화면에서 .vhd 또는 .vhdl 파일을 선택하세요.

자동완성 사용하기

두 글자 이상 입력하면 현재 문서의 선언과 VHDL 항목을 바탕으로 추천이 표시됩니다. 추천을 탭하거나 외장 키보드의 방향키와 Tab 키를 사용해 삽입할 수 있습니다.

검색과 코드 이동

상단 검색 버튼에서 찾기와 바꾸기를 사용할 수 있습니다. 코드 개요에서는 entity, architecture, process, 선언 등으로 이동할 수 있습니다.

자주 묻는 질문

VHDL 코드를 컴파일하거나 시뮬레이션할 수 있나요?

현재 버전은 문서 편집에 집중하며 컴파일이나 시뮬레이션 기능은 제공하지 않습니다.

문서가 외부 서버로 전송되나요?

아니요. Pocket VHDL은 문서 내용을 수집하거나 외부 서버로 전송하지 않습니다. 문서는 사용자가 선택한 로컬 또는 파일 제공자 위치에서 관리됩니다.

지원되는 파일 형식은 무엇인가요?

.vhd와 .vhdl 텍스트 문서를 지원합니다.
