요약

Pocket VHDL은 개인정보를 수집, 저장, 판매 또는 공유하지 않으며 사용자를 추적하지 않습니다.

수집하는 정보

Pocket VHDL과 개발자는 이름, 이메일 주소, 위치, 연락처, 기기 식별자, 사용 기록, 진단 정보 또는 사용자가 작성한 VHDL 문서 내용을 수집하지 않습니다.

문서와 파일

사용자가 생성하거나 여는 .vhd 및 .vhdl 문서는 사용자가 선택한 기기 내 저장소, iCloud Drive 또는 다른 파일 제공자 위치에서 iOS·iPadOS의 문서 기능을 통해 처리됩니다. Pocket VHDL은 문서 내용을 개발자 서버로 전송하지 않습니다. 선택한 파일 제공자의 데이터 처리는 해당 제공자의 정책이 적용될 수 있습니다.

기기 내 설정

글꼴 크기, 탭 너비, 줄 번호 및 자동완성 여부와 같은 편집기 설정은 사용자 편의를 위해 기기에 저장됩니다. 이 설정은 사용자를 식별하는 데 사용되지 않으며 개발자에게 전송되지 않습니다.

추적·광고·분석

Pocket VHDL은 광고 SDK, 사용자 추적 기술 또는 제3자 분석 SDK를 포함하지 않습니다.

지원 페이지와 외부 링크

앱 설정에서 지원 페이지나 개인정보 처리방침을 선택하면 기본 웹 브라우저에서 GitHub Pages 웹사이트가 열립니다. 이 동작은 사용자가 링크를 선택할 때만 발생합니다. 웹 호스팅 제공자인 GitHub는 보안과 서비스 운영을 위해 IP 주소, 브라우저 정보, 요청 시각과 같은 표준 웹 요청 정보를 처리할 수 있으며, 이 경우 GitHub의 개인정보 보호정책이 적용됩니다.

지원 문의

사용자가 이메일로 지원을 요청하면 개발자는 사용자가 자발적으로 제공한 이메일 주소, 문의 내용 및 첨부 파일을 문의 응답과 문제 해결 목적으로 처리합니다. 지원 메일은 Google의 Gmail 서비스를 통해 처리될 수 있습니다. 민감한 소스 코드, 비밀번호 또는 불필요한 개인정보는 보내지 마세요.

보유 및 삭제

지원 문의 정보는 문의 처리, 후속 대응 및 서비스 보호에 필요한 기간 동안만 보유한 뒤 삭제합니다. 자신의 지원 메일과 관련 정보 삭제를 요청하려면 아래 연락처로 문의하세요. 이메일 서비스 제공자의 백업에는 제한된 기간 동안 사본이 남을 수 있습니다.

국외 처리

지원 이메일 또는 웹페이지를 이용하면 GitHub나 Google과 같은 서비스 제공자가 사용자의 국가 밖에서 정보를 처리할 수 있습니다. 해당 제공자의 보호조치와 정책이 적용됩니다. 앱에서 편집하는 VHDL 문서는 개발자가 국외로 전송하지 않습니다.

아동의 개인정보

앱은 어떠한 사용자의 개인정보도 수집하지 않으므로 아동의 개인정보도 의도적으로 수집하지 않습니다.

정책 변경

앱의 데이터 처리 방식이 변경되면 이 방침과 App Store의 개인정보 공개 내용을 업데이트하고 새 시행일을 표시합니다. 중요한 변경은 이 페이지 또는 앱 업데이트 안내를 통해 알립니다.
